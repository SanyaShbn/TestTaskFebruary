# Welcome to TopicHub

In this guide, we'll walk you through how to set up and test the following application locally.

## Installation

First, clone the project repository:

```
https://github.com/SanyaShbn/TestTaskFebruary.git
```

To start docker-compose:

```
docker-compose up
```

## Usage

```
http://localhost:8080
```
